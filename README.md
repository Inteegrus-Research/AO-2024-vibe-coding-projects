🛠️ How to Navigate
Each folder is a standalone project. To run a specific one:

   1. cd <project-name>
   2. Follow the local README.md within that folder.

⚠️ Important Note on Heavy Files
Some projects contain large binaries or media (e.g., .mp4, .exe, node_modules).

* Action Required: Ensure you add a .gitignore to skip node_modules and large video files to keep the repo lean.
